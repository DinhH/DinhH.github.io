Tip of the week.

Tip of the week is actually pretty new to me too. If any of you are using Atom text editor, there's a package called Atom Pair that allows you and your partner to see live changes to the code. You can share your project and any live changes to the project will show for all of you who has the right to the pairing project. This is useful because pair programming is so much easier. However, this is a new package so personally I still think there are some minor bugs. But so far, it has impressed me for this project so far. Definitely worth checking it out.
